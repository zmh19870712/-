项目开发准备工作

1. 开发环境准备

需要准备以下工具：

  1. git
  2. pycharm
  3. python3.6+
  4. linux环境
  5. SecureCRT

1.1 git

git用于代码管理，是企业开发过程中最常用的代码版本管理工具，多个人协同开发一个系统，必然会出现同一个文件同时被多人修改的情况，开发新的功能时，为了不污染原有的代码，也需要建立新的分值，这些都可以使用git来解决。

需要你们做如下准备工作：

  1. 安装git工具
  2. 在github上注册账号，并新建一个项目
  3. 使用git工具从github上拉取代码，并实验其他功能

以下是参考资料：

  1. GitHub如何新建仓库和提交代码
  2. 在github上创建自己的代码仓库
  3. git教程

1.2 pycharm

参考我网站教程安装配置pycharm

1.3 python3.6+

python版本要求是3.6+

1.4 linux环境

linux环境我会提供一台阿里云服务器供大家使用，你们各自发消息给我，给自己起一个账号名称，我设置好密码以后，会分别告诉你们密码

1.5 SecureCRT

SecureCRT可用于连接阿里云服务器，你有其他连接工具也可以用
